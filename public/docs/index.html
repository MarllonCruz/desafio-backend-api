Make a transaction in the wallet

Example request:
curl --request POST \
    "https://localhost/api/api/transactions?title=veritatis&value=7&type=eveniet" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "https://localhost/api/api/transactions"
);

const params = {
    "title": "veritatis",
    "value": "7",
    "type": "eveniet",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "POST",
    headers,
}).then(response => response.json());

Example response (200):


{
    "id": "uuid",
    "title": "Pagamento da fatura",
    "value": 500,
    "type": "income"
}

Request      

POST api/transactions

Query Parameters

title  string  

Title for transaction

value  integer  

Value for transaction

type  string  

Type field must be "income" or "outcome"

Get all transactions and total amount in balance

Example request:
curl --request GET \
    --get "https://localhost/api/api/transactions" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "https://localhost/api/api/transactions"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
